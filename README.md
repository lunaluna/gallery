# USAGE

## preparering

```
$ php --version
```

Uses the built-in server in PHP 5.4.0+.

## for update

```
$ curl -s http://php-osx.liip.ch/install.sh | bash -s 5.5
```

### update .bashrc

```
export PATH=/usr/local/php5/bin:$PATH
```

### then reload

```
$ source .bashrc
```

## php server use

```
$ cd path/to/gulp/file/directory
$ npm i (if you run first time)
$ npm start
```
